### Backend Development
1. Create git repo
   git init

2. add .gitignore file

3. create .env file for environment variables & read data from .env
    with "dotenv" (npm i dotenv)

4. Generate Package.json (npm init -y)

5. Create express app (npm i express)

6. Connect to Database (npm i mongoose)

7. Add middlewares
   body parser, error handling middlewares

8. Design Schema and create models

9. Design REST APIs for all resources
